## 2. 参考書の構造分析

### 2.1 全体構成

高校英文法を15のPartに体系的に配列した解説書。各Partの下に2〜23個の小セクション（計162）がある。セクション単位が非常に細かく、1セクションは2〜3ページ程度。

```
Part 01 > 動詞の語法と文型（23セクション）
Part 02 > 受動態（2セクション）
Part 03 > 時制（10セクション）
Part 04 > 助動詞（12セクション）
Part 05 > 仮定法（9セクション）
Part 06 > 準動詞（17セクション）
Part 07 > 接続詞（14セクション）
Part 08 > 関係詞（13セクション）
Part 09 > 前置詞（20セクション）
Part 10 > 比較（12セクション）
Part 11 > 冠詞・名詞（7セクション）
Part 12 > 代名詞（6セクション）
Part 13 > 形容詞（9セクション）
Part 14 > 副詞（4セクション）
Part 15 > 特殊構文・その他（4セクション）
```

### 2.2 1セクションの構造パターン

セクションは2〜3ページと短く、以下の統一パターンを持つ。

```
パターン（全セクション共通）:

┌─ 導入 ──────────────────────────────┐
│  会話調の問題提起（教師/生徒の対話や                │
│  「〜ではありません！」式のキャッチコピー）           │
└────────────────────────────────────────┘

┌─ 本編解説 ───────────────────────────────┐
│  Ⓐ 小見出し1                                    │
│    解説文 + 番号付き例文（001, 002, ...）           │
│    + イラスト・図解 + 黄色ハイライト(重要ルール)      │
│  Ⓐ/Ⓑ 小見出し2                                  │
│    ...                                          │
│  Ⓒ/Ⓓ 小見出し3（あれば）                          │
│    ...                                          │
│  （表・分類・まとめ）                               │
└────────────────────────────────────────┘

┌─ プラスα（脚注） ───────────────────────┐
│  ＊1, ＊2, ... 補足説明・参照先                     │
│  別冊参照（→ 別 p.XX のポイントXXX）              │
└────────────────────────────────────────┘
```

**本冊と別冊の対応関係**:

```
本冊セクション             別冊
┌─────────────┐     ┌─────────────────────────┐
│ Ⓐ 解説       │     │ 深める！英文法ポイント XXX        │
│ → 別p.XX参照  │──→ │  ・ルールの凝縮版                  │
│ 番号付き例文   │     │  ・□ 動詞/形容詞/表現リスト        │
│ イラスト      │     │  ・▶ 追加例文                      │
│ Ⓑ 解説       │     │ 深める！英文法ポイント YYY        │
│ → 別p.YY参照  │──→ │  ・表形式の比較・整理               │
└─────────────┘     └─────────────────────────┘
```

### 2.3 特殊な構造要素

| 要素 | 説明 | 保存方針 |
|------|------|---------|
| **Ⓐ Ⓑ Ⓒ Ⓓ 小見出し** | 本編解説の区切り（丸囲み文字）。ノード抽出の自然な単位。大文字・小文字(ⓐⓑⓒⓓ)の両方が使われるが構造上の差異なし | Ⓐ ごとにノード化を検討 |
| **番号付き例文** | `**001**` 〜 `**8XX**` の通し番号。書籍全体で連番。和訳付きの完全な英文 | sentences に登録（role: example） |
| **黄色ハイライト** | `<span style="background:yellow">重要ルール</span>` 形式のキーポイント | knowledge_nodes の check_points または understanding_goals に抽出 |
| **イラスト・図解** | `[図表: 説明]` 形式で書き起こし済み。概念の視覚化 | notes に `[図表: 種類 — 内容要約（p.XX）]` と記述 |
| **プラスα 脚注** | ＊1, ＊2 等の補足説明。他セクション・別冊への参照を含む | 重要なものは notes に記述。参照は「参考書内参照: →p.XX」形式 |
| **別冊参照** | `→ 別 p.XX のポイントXXX` 形式の相互参照 | notes に「別冊ポイントXXX参照」と記述 |
| **教師/生徒の対話** | 導入部の会話調説明 | ノード化しない（教育的導入。知識そのものではない） |
| **誤文 (✕)** | 誤用例（`(✕) alive animals` 等） | 関連ノードの check_points に「✕ 〜は不可」として記録 |
| **別冊「深める！英文法ポイント」** | 全356項目。ルールの凝縮版＋動詞/表現リスト＋追加例文 | ノードの check_points・notes の**主要な情報源**として活用。別冊例文は sentences には登録しない |
| **別冊「□リスト」** | □ go bad, □ go bankrupt 等の語彙チェックリスト | 関連ノードの check_points に変換。大規模リストは代表例のみ |
| **▶ 別冊内例文** | ▶ マーク付き例文（別冊内の補足例文） | sentences には登録しない（本冊の番号付き例文のみ登録） |

### 2.4 セクションの種類

162セクションには以下の**3タイプ**がある:

| タイプ | 説明 | 例 | 概数 |
|--------|------|-----|------|
| **概念解説型** | 文法概念の「なぜ」を解説。核心的な理解を促す | 現在形、直説法と仮定法、関係詞の働き | ~80 |
| **パターン・語法型** | 動詞パターンや類義語の使い分けを解説 | パターンで理解する動詞(1)、「言う」の動詞 | ~40 |
| **重要表現型** | 慣用表現・構文のまとめ。リスト要素が多い | mayを用いた重要表現、動名詞を用いた重要表現 | ~40 |

### 3.2 ID体系

| 対象 | 形式 | 例 |
|------|------|---|
| **knowledge_nodes** | `{カテゴリ接頭辞}-{3桁連番}` | `nrv-001`, `nrt-015` |
| **sections** | `Nar_{Part2桁}_{セクション2桁}` | `Nar_01_01`, `Nar_09_17` |
| **sentences（番号付き例文）** | `nar-{Part2桁}{セクション2桁}-e{書籍通し番号}` | `nar-0101-e001`, `nar-0910-e510` |
| **sentences（▶パターン例文）** | `nar-{Part2桁}{セクション2桁}-v{セクション内連番}` | `nar-0301-v001` |

#### sentences ID の補足

- `e` = 番号付き例文。書籍全体の通し番号（001〜8XX）をそのまま使用し、原本との照合を容易にする
- `v` = ▶パターン例文（variant）。書籍通し番号がないため、セクション内連番（001〜）を振る
- role は `example` のみ（本書にはCHECK・入試問題セクションがない）

#### ノード統合の基準

この参考書はセクションが細かいため、**複数セクションが同一ノードを共有する場合がある**:

- **統合する**: 連番セクション（例: Part03_03「過去形と現在完了形(1)」+ Part03_04「(2)」）が同一概念の前後編の場合
- **分割しない**: セクション内の複数 Ⓐ が同一概念の異なる側面を扱う場合（上記の具体例参照）
- **分割する**: セクション内に明確に独立テスト可能な複数概念がある場合（稀）

### 3.4 粒度基準

| 指標 | 目安範囲 | 根拠 |
|------|---------|------|
| ノード/セクション比率 | **1.0〜2.0** | セクションが2〜3pと既に非常に細かいため、共通基準(2.0〜4.0)より低い |
| check_points/ノード | 3〜5 | 共通基準 |
| understanding_goals/ノード | 3〜6 | 共通基準 |

**想定ノード数**: 150〜220ノード（162セクション × 平均1.0〜1.4ノード）

**粒度基準が他参考書と異なる理由**: 核心(26セクション×7〜36p)や入門英文(85セクション×数p)と比べ、本書は162セクション×2〜3pと著しく細粒度。セクションが既にノード相当の粒度を持つため、1セクション1ノードが自然な場合が多い。

**分割判断基準**（共通）:
1. 独立テスト可能性: その小見出しの内容だけで確認テストの問題を作れるか？
2. 部分習得可能性: 一方を知っていて他方を知らない状態がありえるか？
3. 十分な関連英文: そのノードに紐づく英文が2文以上あるか？（本書のセクションは例文が少ないため基準を3→2に緩和）

### 4.3 構文分析の方針

本書には詳細な構文記号は使われていないが、文型表記（S V O C M）が頻出する。これを活用する。

```yaml
sentence_structures:
  - sentence_id: "nar-0101-e001"
    label: "overall"
    value: "S V M M"

  - sentence_id: "nar-0101-e001"
    label: "details"
    value: "<She>(S) lives(V) [happily](M:副詞) [in Italy](M:前置詞句)"
```

**label の種類**:
- `overall`: 文型記号（S V O C M）— 本書が頻繁に使用
- `details`: 句・節のカタマリと役割の分析

## 5. セクション設計

### 5.1 セクション分割方針

- **原則: 1書籍セクション = 1システムセクション**（分割・統合なし）

- 連番セクション（(1)(2)）でも統合しない — 本書の著者がページ数の制約で分けた構成を尊重
- **合計: 162セクション**

### 5.4 セクション間の前提関係（section_prerequisites）

本書はセクションが非常に細かく、特に**連番セクション（(1)(2)）は必ず前後関係がある**。

#### 前提関係の原則

1. **連番セクション**: (1) → (2) → (3) は必ず前提関係を設定
2. **概要セクション → 詳細セクション**: Part内の最初のセクション（概要説明）は後続の前提
3. **Cross-Part依存**: 原則なし（各Partはほぼ独立）。ただし、Part01（文型）の基本はPart02以降の前提知識
